# <a name="certified-consulting-partners-to-deliver-mobile-solutions"></a>モバイル ソリューションを提供するコンサルティング パートナーの認定

> [!TIP]
> **Xamarin のコンサルティング パートナー プログラムには、2018 年 6 月 30 日の時点で、Microsoft Partner Network にマージします。**
>
> 概要、 [Microsoft Partner Network](https://partner.microsoft.com/):
>
> * わからない場合既に登録済みの Microsoft Partner Network メンバー[登録](https://partner.microsoft.com/membership)パートナーになります。  
> * 実行して、専門知識を示す、[アプリケーション開発のコンピテンシー](https://partner.microsoft.com/membership/application-development-competency)と[Cloud Platform コンピテンシー](https://partner.microsoft.com/membership/cloud-platform-competency)します。
> * として探索可能なので、Xamarin、Android、iOS、Mobile Apps、およびモバイル アプリケーション開発を含めるのプロファイルの専門分野のタグを更新する、 [Microsoft ソリューション プロバイダー](https://www.microsoft.com/solution-providers/home)します。

Visual Studio と Xamarin では、ランプアップのモバイル、ビジネスを実行するためのコンサルティング パートナーを見つけて、品質の高いモバイル ソリューションを構築します。 長期的な戦略的パートナーとは、アプリのライフ サイクルのすべての段階を説明できるし、設計、開発、およびクロス プラットフォーム モバイル アプリの高いパフォーマンスをテストする際に役立つモバイルの専門家です。

_プライマリの場所で表示されている、多くのパートナーは、国境を越えてサービスを提供します。_
