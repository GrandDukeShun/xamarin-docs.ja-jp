# <a name="native-types-for-ios-and-macos"></a>IOS および macOS ネイティブ型

Mac および iOS API は、アーキテクチャに固有のデータ型を使用します。32 ビット プラットフォームでは常に 32 ビットであり、64 ビット プラットフォームでは常に 64 ビットとなります。
